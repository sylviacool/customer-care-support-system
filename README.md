# Customer Care Support System

This project is a console-based customer support application for an electronic gadgets company.

Customers can contact different departments based on their issue.

## Departments
- Payment Department
- Query Department
- Sales Department

## Features
- Customer selects a department
- Customer submits an issue
- Department registers the issue
- Console-based interaction

## Technologies Used
- Java
- Spring Framework (XML configuration)
- Maven

## Concepts Demonstrated
- Interfaces
- Polymorphism
- Dependency Injection
- Spring Bean Configuration
