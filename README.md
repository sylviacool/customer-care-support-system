# Customer Care Support System

This project simulates a customer support system for an electronic gadgets company.

Customers can register issues with different departments, while support staff can review all logged issues through the console application.

## Departments

- Payment Department
- Query Department
- Sales Department

## Features

- Customer issue registration
- Department selection via console
- Issue tracking with generated reference ID
- View all registered issues (staff view)
- Total issue count display
- Continuous system loop for multiple users

## Technologies Used

- Java
- Spring Framework (XML configuration)
- Maven

## Concepts Demonstrated

- Interfaces
- Polymorphism
- Dependency Injection
- Spring Bean Configuration
- Lists and loops
- Console input handling
